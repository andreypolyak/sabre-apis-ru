Алгоритм обработки очередей

Схема

Получение списка очередей (QueueCountLLSRQ)

Для получения списка очередей используется сервис QueueCountLLSRQ.

По умолчанию запрашивается список очередей для текущего PCC, однако в атрибуте /QueueCountRQ/QueueInfo/QueueIdentifier/@PseudoCityCode можно задать другой PCC (требуется наличие Branch Access).

Пример запроса
Копировать 
<QueueCountRQ ReturnHostCommand="true" Version="2.1.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <QueueInfo>
    <QueueIdentifier PseudoCityCode="9LSC"/>
  </QueueInfo>
</QueueCountRQ>

Пример ответа
Копировать 
<QueueCountRS Version="2.1.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2016-11-03T16:02:41-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="38D40B">QC/9LSC</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
  <QueueInfo DateTime="11-04T00:02" PseudoCityCode="9LSC">
    <QueueIdentifier Count="3" Number="S"/>
    <QueueIdentifier Count="5" Number="1"/>
    <QueueIdentifier Count="1" Number="6"/>
    <QueueIdentifier Count="2" Number="7"/>
    <QueueIdentifier Count="4" Number="9"/>
    <QueueIdentifier Count="27" Number="96"/>
    <QueueIdentifier Count="2" Number="97"/>
    <QueueIdentifier Count="1" Number="98"/>
  </QueueInfo>
  <Totals Count="3" Type="MESSAGES"/>
  <Totals Count="0" Type="SPECIALS"/>
  <Totals Count="42" Type="PNRS"/>
</QueueCountRS>

Получение списка бронирований в очереди (QueueAccessLLSRQ)

Для получения списка бронирований в очереди используется сервис QueueAccessLLSRQ. Получение списка бронирований в очереди — информационный запрос, после него не происходит вход в очередь, а значит бронированиями в этой очереди нельзя управлять.

В запросе должны быть указаны:

  • /QueueAccessRQ/QueueIdentifier/@Number — номер очереди
  • /QueueAccessRQ/QueueIdentifier/@PseudoCityCode — PCC, в котором находится очередь
  • /QueueAccessRQ/QueueIdentifier/List/@Ind — индикатор получения списка бронирований в очереди (значение true)

Пример запроса
Копировать 
<QueueAccessRQ ReturnHostCommand="true" Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <QueueIdentifier Number="6" PseudoCityCode="9LSC">
    <List Ind="true"/>
  </QueueIdentifier>
</QueueAccessRQ>

Пример ответа
Копировать 
<QueueAccessRS Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2016-11-03T16:06:44-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="38D40B">Q/9LSC7/L</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
  <Line Number="1">
    <DateTime>2016-08-07T15:01</DateTime>
    <POS>
      <Source AgentSine="M1A" PseudoCityCode="PLT"/>
    </POS>
    <UniqueID ID="CZDMWD"/>
  </Line>
  <Line Number="2">
    <DateTime>2016-10-30T19:01</DateTime>
    <POS>
      <Source AgentSine="MSU" PseudoCityCode="PLT"/>
    </POS>
    <UniqueID ID="PKYDIL"/>
  </Line>
  <Paragraph>
    <Text>QUEUE LIST FOR 9LSC  QUEUE 007 ON 04NOV AT 0006</Text>
    <Text>NUM PNR LOCATOR    PLACED BY     DATE      TIME    GROUP</Text>
    <Text>1  CZDMWD        PLT   M1A     07AUG16   1501</Text>
    <Text>2  PKYDIL        PLT   MSU     30OCT16   1901</Text>
    <Text>END OF LIST</Text>
  </Paragraph>
</QueueAccessRS>

Чтение бронирования (TravelItineraryReadRQ)

Для чтения бронирования используется сервис TravelItineraryReadRQ.

В качестве значения атрибута /TravelItineraryReadRQ/UniqueID/@ID в запросе указывается код бронирования (PNR Record Locator).

Пример запроса
Копировать 
<TravelItineraryReadRQ Version="3.8.0" xmlns="http://services.sabre.com/res/tir/v3_8">
  <MessagingDetails>
    <SubjectAreas>
      <SubjectArea>FULL</SubjectArea>
    </SubjectAreas>
  </MessagingDetails>
  <UniqueID ID="CZDMWD"/>
</TravelItineraryReadRQ>

Редактирование бронирования

Текущее открытое бронирование можно изменить. См. Редактирование бронирований, Оформление билетов и EMD, Отмена бронирований, Войдирование билетов и EMD и др.

Удаление бронирования из очереди (QueueAccessLLSRQ)

Для удаления бронирования из очереди используется сервис QueueAccessLLSRQ.

В запросе необходимо указать:

  • /QueueAccessRQ/Navigation/@Action — тип операции (значение QR)
  • /QueueAccessRQ/QueueIdentifier/@PseudoCityCode — PCC
  • /QueueAccessRQ/QueueIdentifier/@Number — номер очереди

Пример запроса
Копировать 
<QueueAccessRQ ReturnHostCommand="true" Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <Navigation Action="QR"/>
  <QueueIdentifier PseudoCityCode="9LSC" Number="7"/>
</QueueAccessRQ>

Пример ответа
Копировать 
<QueueAccessRS Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2016-11-03T16:09:15-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="38D40B">QR/9LSC7</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
</QueueAccessRS>

Сохранение бронирования (EndTransactionLLSRQ)

Для сохранения изменений в бронировании необходимо отправить запрос к сервису EndTransactionLLSRQ. В результате выполнения запроса все изменения в бронировании, выполненные в текущей сессии, будут сохранены, а бронирование закрыто.

В запросе необходимо указать:

  • /EndTransactionRQ/EndTransaction/@Ind — признак сохранения бронирования (значение true)
  • /EndTransactionRQ/Source/@ReceivedFrom — значения поля Received From. Используется для идентификации инициатора изменений в истории бронирования

Пример запроса
Копировать 
<EndTransactionRQ ReturnHostCommand="true" Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <EndTransaction Ind="true"/>
  <Source ReceivedFrom="SWS"/>
</EndTransactionRQ>

Пример ответа
Копировать 
<EndTransactionRS Version="2.0.8" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2016-11-03T16:09:57-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="38D40B">6SWS§E</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
  <ItineraryRef ID="CZDMWD">
    <Source CreateDateTime="2016-11-03T16:09"/>
  </ItineraryRef>
  <Text>OK 1609 CZDMWD</Text>
</EndTransactionRS>
