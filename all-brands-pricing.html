Расчет стоимости по всем брендам

Для получения расчета стоимости выбранного перелета по всем доступным брендам используется сервис PO_AllBrandsPricingRQ.

Сервис не требует наличия созданного бронирования, а вся информация о перелете должна быть указана в запросе:

  • /AllBrandsPricingRQ/AirItinerary/OriginDestinationOptions/OriginDestinationOption/FlightSegment — сегменты:
    • /@SegmentNumber — номер сегмента в бронировании
    • /@SegmentType — тип сегмента (авиасегмент — A, наземный сегмент (ARNK) — K)
    • /@FlightNumber — номер рейса
    • /@DepartureDate — дата и время отправления
    • /@ArrivalDate — дата и время прибытия
    • /@ResBookDesigCode — класс бронирования
    • /DepartureAirport/@LocationCode — аэропорт отправления
    • /ArrivalAirport/@LocationCode — аэропорт прибытия
    • /MarketingAirline/@Code — код маркетингового перевозчика
    • /Marriage/@Status — статус соединенного сегмента (первый сегмент в цепочке соединенных сегментов — S, средний сегмент в цепочке соединенных сегментов — P, последний сегмент в цепочке соединенных сегментов — E) Для не соединенных сегментов элемент не добавляется
  • /AllBrandsPricingRQ/PriceRequestInformation/PassengerTypes/PassengerType — пассажиры:
    • /@Code — категория пассажиров
    • /@Count — количество пассажиров данной категории

Опционально в элементе /AllBrandsPricingRQ/PriceRequestInformation запроса можно указать следующие параметры расчета стоимости:

  • /Account/@Code — аккаунт код (Account Code)
  • /Corporate/@ID — код корпоративной скидки (Corporate ID)
  • /ForceCorporate/@Ind — расчет стоимости только по тарифам с указанными выше аккаунт кодами и кодами корпоративной скидки (значение true)
  • /PublicFare/@Ind — расчет стоимости только по публичным тарифам (значение true)
  • /PrivateFare/@Ind — расчет стоимости только по приватным тарифам (значение true)
  • /ValidatingCarrier/@Code — код валидирующего перевозчика для выполнения расчета стоимости

<AllBrandsPricingRQ Version="1.0.0" xmlns="http://webservices.sabre.com/sabreXML/2003/07">
  <PriceRequestInformation>

      <PassengerType Code="ADT" Count="2"/>
      <PassengerType Code="CNN" Count="1"/>
      <PassengerType Code="INF" Count="1"/>
    </PassengerTypes>
  </PriceRequestInformation>
  <AirItinerary>
    <OriginDestinationOptions>
      <OriginDestinationOption>
        <FlightSegment DepartureDate="2017-09-01T18:00:00" ArrivalDate="2017-09-01T19:25:00" FlightNumber="80" ResBookDesigCode="W" SegmentNumber="1" SegmentType="A">
          <DepartureAirport LocationCode="LED"/>
          <ArrivalAirport LocationCode="DME"/>
          <MarketingAirline Code="U6"/>
        </FlightSegment>
      </OriginDestinationOption>
      <OriginDestinationOption>
        <FlightSegment DepartureDate="2017-09-01T22:00:00" ArrivalDate="2017-09-02T02:20:00" FlightNumber="265" ResBookDesigCode="W" SegmentNumber="2" SegmentType="A">

          <MarketingAirline Code="U6"/>
        </FlightSegment>
      </OriginDestinationOption>
    </OriginDestinationOptions>
  </AirItinerary>
</AllBrandsPricingRQ>
