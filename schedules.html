Получение расписаний

Для получения расписаний используется сервис OTA_AirScheduleLLSRQ.

Для получения расписания рейсов необходимо указать в запросе:

  • /OTA_AirScheduleRQ/OptionalQualifiers/FlightQualifiers/VendorPrefs/Airline/@Code — код маркетингового перевозчика
  • /OTA_AirScheduleRQ/OriginDestinationInformation/FlightSegment/@DepartureDateTime — дата отправления
  • /OTA_AirScheduleRQ/OriginDestinationInformation/FlightSegment/OriginLocation/@LocationCode — код пункта отправления
  • /OTA_AirScheduleRQ/OriginDestinationInformation/FlightSegment/DestinationLocation/@LocationCode — код пункта назначения

Дополнительно в запросе можно указать:

  • /OTA_AirScheduleRQ/OptionalQualifiers/FlightQualifiers/@DirectOnly — значение true позволит искать только прямые рейсы
  • /OTA_AirScheduleRQ/OptionalQualifiers/FlightQualifiers/@ExcludeCodeshares — значение true позволит исключить кодшейринговые рейсы
  • /OTA_AirScheduleRQ/OptionalQualifiers/TimeQualifiers/DepartureTime/@WindowAfter и @WindowBefore — показать расписание для рейсов с вылетом в указанный временной промежуток
  • /OTA_AirScheduleRQ/OriginDestinationInformation/FlightSegment/ConnectionLocations/ConnectionLocation/@LocationCode — код пункта пересадки
  • /OTA_AirScheduleRQ/OriginDestinationInformation/FlightSegment/ConnectionLocations/ConnectTimeOverrides/@Min и @Max — указать минимальное и максимальное время стыковки

Пример запроса
Копировать 
<OTA_AirScheduleRQ ReturnHostCommand="true" Version="2.1.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <OptionalQualifiers>
    <FlightQualifiers DirectOnly="true">
      <VendorPrefs>
        <Airline Code="SU"/>
      </VendorPrefs>
    </FlightQualifiers>
  </OptionalQualifiers>
  <OriginDestinationInformation>
    <FlightSegment DepartureDateTime="12-31T00:00">
      <DestinationLocation LocationCode="LED"/>
      <OriginLocation LocationCode="MOW"/>
    </FlightSegment>
  </OriginDestinationInformation>
</OTA_AirScheduleRQ>
