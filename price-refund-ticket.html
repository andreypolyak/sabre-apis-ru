Расчет стоимости возврата билетов

Расчет стоимости возврата билетов

Оглавление:

  • Введение
  • Алгоритм расчета стоимости возврата билетов
  • Переход в другой PCC (ContextChangeLLSRQ)
  • Расчет стоимости возврата билета (TKT_RefundRQ)

Введение

Возврат билетов при работе через Sabre APIs осуществляется при помощи дополнительного продукта Sabre Automated Refunds, который автоматизирует процесс расчета и выполнения возврата. Для подключения продукта, пожалуйста, обратитесь к вашему куратору в Sabre. Продукт позволяет вернуть любой билет при соблюдении двух следующих условий:

  1. Тариф (или тарифы), по которым был оформлен билет содержит 33-ю категорию правил.
  2. Перевозчик входит в список перевозчиков, разрешающих выполнение автоматических возвратов.

В данном разделе описан расчет стоимости возврата билетов. Процесс возврата билетов описан в разделе Возврат билетов.

Переход в другой PCC (ContextChangeLLSRQ)

Возврат билетов необходимо производить в том PCC, где они были изначально оформлены. В случае, если билеты не были оформлены в iPCC, то необходимо перейти в другой PCC, отправив запрос к сервису ContextChangeLLSRQ.

Обратите внимание на то, что для перехода в другой PCC требуется наличие Branch Access между ним и iPCC, в котором была создана сессия. Подробнее о Branch Access см. разделе Конфигурация Sabre.

В качестве значения атрибута /ContextChangeRQ/ChangeAAA/@PseudoCityCode в запросе указывается PCC, в котором будет произведен обмен.

Пример запроса
Копировать 
<ContextChangeRQ ReturnHostCommand="true" Version="2.0.3" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <ChangeAAA PseudoCityCode="2FRH"/>
</ContextChangeRQ>

Пример ответа
Копировать

  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2018-04-03T06:21:34-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="163133">AAA2FRH*SWS-OVER*</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
  <SecurityToken Updated="false">Shared/IDL:IceSess\/SessMgr:1\.0.IDL/Common/!ICESMS\/ACPCRTD!ICESMSLB\/CRT.LB!-3161524137492458618!1896427!0</SecurityToken>
  <Text>2FRH.9LSC*AWS.A</Text>

Расчет стоимости возврата билета (TKT_RefundRQ)

Для расчета стоимости возврата билета используется сервис TKT_RefundRQ. Обратите внимание на то, что расчет стоимости возврата билета является бесплатной операцией. Вызов сервиса для расчета стоимости возврата билета может использоваться также для проверки возможности автоматического возврата билета через API.

В запросе необходимо указать:

  • /RefundRQ/POS/STL:Company — код системы бронирования. Всегда 1S
  • /RefundRQ/POS/STL:Pseudo — PCC, в котором производится возврат билета
  • /RefundRQ/TransactionInfo/@requestType — тип запроса. Всегда Price
  • /RefundRQ/TransactionInfo/TicketingDocument/Number — номер билета (13 цифр)

Пример запроса
Копировать 
<RefundRQ version="1.1.0" xmlns="http://www.sabre.com/ns/Ticketing/ExchangeRefund/1.0" xmlns:STL="http://services.sabre.com/STL/v01">
  <POS>
    <STL:Company>1S</STL:Company>
    <STL:Pseudo>2FRH</STL:Pseudo>
  </POS>
  <TransactionInfo requestType="Price">
    <TicketingDocument>
      <Number>6285301772440</Number>
    </TicketingDocument>
  </TransactionInfo>
</RefundRQ>
