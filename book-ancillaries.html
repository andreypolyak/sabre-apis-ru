Бронирование услуг

Бронирование дополнительных услуг

Для бронирования дополнительных услуг и отправки SSR используется сервис UpdateReservationRQ.

Параметры запроса

В запросе необходимо указать:

  • /UpdateReservationRQ/RequestType/@commitTransaction — признак сохранения бронирования после выполнения запроса. Всегда значение true
  • /UpdateReservationRQ/RequestType/@initialIgnore — признак игнорирования бронирования перед выполнением запроса. Всегда значение true
  • /UpdateReservationRQ/RequestType — тип запроса. Всегда значение Stateless
  • /UpdateReservationRQ/ReturnOptions/@IncludeUpdateDetails — признак получения деталей изменения бронирования в ответе. Всегда значение true
  • /UpdateReservationRQ/ReturnOptions/@RetrievePNR — признак получения состояния бронирования в ответе. Всегда значение true
  • /UpdateReservationRQ/ReservationUpdateList/Locator — код бронирования
  • /UpdateReservationRQ/ReservationUpdateList/ReceivedFrom/AgentName — значения поля Received From. Используется для идентификации инициатора изменений в истории бронирования

Бронирование дополнительных услуг

Для каждой бронируемой дополнительной услуги необходимо указать (в скобках обозначен соответствующий элемент из ответа на запрос к сервису GetAncillaryOffersRQ):

  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/@op — код типа операции. Всегда значение C (Create, создание)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/NameAssociationList — информация о пассажирах (одна услуга может быть забронирована сразу для нескольких пассажиров в том случае, если они имеют одинаковую категорию пассажира, а также количество требуемых услуг равно):
    • /NameAssociationTag/LastName — фамилия пассажира
    • /NameAssociationTag/FirstName — имя пассажира
    • /NameAssociationTag/NameRefNumber — номер пассажира
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/SegmentAssociationList — информация о сегментах:
    • /SegmentAssociationTag/CarrierCode — маркетинговый перевозчик
    • /SegmentAssociationTag/FlightNumber — номер рейса
    • /SegmentAssociationTag/DepartureDate — дата вылета
    • /SegmentAssociationTag/BoardPoint — аэропорт отправления
    • /SegmentAssociationTag/OffPoint — аэропорт прибытия
    • /SegmentAssociationTag/ClassOfService — класс бронирования
    • /SegmentAssociationTag/BookingStatus — статус сегмента
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/CommercialName — название услуги (/GetAncillaryOffersRS/AncillaryDefinition/CommercialName)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/RficCode — RFIC код (/GetAncillaryOffersRS/AncillaryDefinition/ReasonForIssuance/@code)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/RficSubcode — RFIC субкод (/GetAncillaryOffersRS/AncillaryDefinition/SubCode)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/SSRCode — код требуемого SSR сообщения (/GetAncillaryOffersRS/AncillaryDefinition/SpecialService)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/OwningCarrierCode — код перевозчика, продающего услугу (/GetAncillaryOffersRS/AncillaryDefinition/Airline)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/BookingIndicator — индикатор бронирования (/GetAncillaryOffersRS/AncillaryDefinition/BookingMethod/@code)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/Vendor — код системы, в которой размещена дополнительная услуга (/GetAncillaryOffersRS/AncillaryDefinition/Vendor)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/EMDType — тип EMD (/GetAncillaryOffersRS/AncillaryDefinition/ElectronicMiscDocType/@code)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/ServiceType — тип услуги (/GetAncillaryOffersRS/Ancillary/ServiceType)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/EquivalentPrice/Price — стоимость услуги в валюте эквивалента (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/EquivAmount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/EquivalentPrice/Currency — валюта эквивалента (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/EquivAmount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TTLPrice/Price — общая стоимость услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/TTL_Price/EquivAmount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TTLPrice/Currency — валюта общей стоимости услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/TTL_Price/EquivAmount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/OriginalBasePrice/Price — оригинальная стоимость услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/Amount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/OriginalBasePrice/Currency — валюта оригинальной стоимости услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/Amount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/RefundIndicator — индикатор возвратности (/GetAncillaryOffersRS/Offers/AncillaryRules/RefundableReissuable)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/CommisionIndicator — индикатор наличия комиссии (/GetAncillaryOffersRS/Offers/AncillaryRules/Commissionable)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/InterlineIndicator — индикатор интерлайна (/GetAncillaryOffersRS/Offers/AncillaryRules/Interlineable)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/FeeApplicationIndicator — индикатор применимости оплаты (/GetAncillaryOffersRS/Offers/AncillaryRules/FeeApplicationMethod)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/PassengerTypeCode — категория пассажира (/GetAncillaryOffersRS/PassengerOffers/PassengerReference/@passengerType)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/BoardPoint — аэропорт отправления для услуги (/GetAncillaryOffersRS/Offers/Origin)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/OffPoint — аэропорт прибытия для услуги (/GetAncillaryOffersRS/Offers/Destination)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/SimultaneousTicketIndicator — индикатор необходимости одновременного оформления билета (/GetAncillaryOffersRS/Offers/AncillaryRules/SimultaneousTicketIndicator)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TravelDateEffective — дата окончания действия предложения (/GetAncillaryOffersRS/Ancillary/TravelDateEffective)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/LatestTravelDatePermitted — дата начала действия предложения (/GetAncillaryOffersRS/Ancillary/LatestTravelDatePermitted)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/PurchaseByDate — дата необходимости оформления EMD для дополнительной услуги (/GetAncillaryOffersRS/Offers/PurchaseByDate)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalEquivalentPrice/Price — общая стоимость всех услуг выбранного типа в валюте эквивалента (количество бронируемых услуг умножить на /GetAncillaryOffersRS/Offers/AncillaryFee/Base/EquivAmount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalEquivalentPrice/Currency — валюта эквивалента (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/EquivAmount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalTTLPrice/Price — общая стоимость стоимость всех услуг выбранного типа в валюте эквивалента (количество бронируемых услуг умножить на /GetAncillaryOffersRS/Offers/AncillaryFee/TTL_Price/EquivAmount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalTTLPrice/Currency — валюта общей стоимости услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/TTL_Price/EquivAmount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalOriginalBasePrice/Price — общая оригинальная стоимость всех услуг выбранного типа в валюте эквивалента (количество бронируемых услуг умножить на /GetAncillaryOffersRS/Offers/AncillaryFee/Base/Amount)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/TotalOriginalBasePrice/Currency — валюта оригинальной стоимости услуги (/GetAncillaryOffersRS/Offers/AncillaryFee/Base/Amount/@currency)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/NumberOfItems — количество бронируемых услуг
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/SegmentIndicator — индикатор применимости услуги к сегментам или части бронирования (/GetAncillaryOffersRS/Ancillary/SectorPortionInd)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/RefundFormIndicator — индикатор возвратности (/GetAncillaryOffersRS/Offers/AncillaryRules/FormOfRefund/@code)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/GroupCode — код категории услуги (/GetAncillaryOffersRS/AncillaryDefinition/Group)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/AncillaryServicesUpdate/PriceMismatchAction — действие при несовпадении стоимости услуги при бронировании. Всегда значение ACCEPT_ANY_PRICE

Отправка SSR

Для получения подтверждения некоторых дополнительных услуг требуется отправка SSR сообщений. Список форматов SSR сообщений для некоторых перевозчиков доступен в таблице на портале Dev Studio, на ресурсе Format Finder, а также на сайтах и в рассылках перевозчиков.

Для отправки SSR необходимо указать в запросе:

  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/@op — код типа операции. Всегда значение C (Create, создание)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/@type — тип SSR. Значение H для перевозчика American Airlines (AA), значение G — для всех остальных
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/NameAssociationList/NameAssociationTag/NameRefNumber — номер пассажира
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SpecialService/Code — код SSR сообщения (4 буквы)
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SpecialService/Text — текст SSR сообщения

Опционально в запросе можно указать сегменты, к которым необходимо привязать SSR сообщение. Для этого для каждого сегмента необходимо указать:

  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/CarrierCode — маркетинговый перевозчик
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/FlightNumber — номер рейса
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/DepartureDate — дата вылета
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/BoardPoint — аэропорт отправления
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/OffPoint — аэропорт прибытия
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/ClassOfService — класс бронирования
  • /UpdateReservationRQ/ReservationUpdateList/ReservationUpdateItem/SpecialServiceRequestUpdate/SegmentAssociationTag/BookingStatus — статус сегмента

Обратите внимание на то, что для дополнительных улуг обязательно требуется оформление EMD. EMD должны быть оформлены в бронировании после оформления билетов. Процесс оформления EMD подробно описан в разделе Оформление билетов и EMD.
