Проверка наличия мест

Проверка наличия мест

Оглавление:

  • Получение информации о наличии мест для определенного рейса
  • Получение информации о наличии мест для всех рейсов по заданному направлению
  • Получение информации о наличии мест для дополнительных рейсов

Для получения информации о наличии мест используется сервис OTA_AirAvailLLSRQ.

Не рекомендуется использовать этот сервис в стандартном процессе бронирования из результатов поиска для проверки наличия мест для выбранного варианта перелета!

Существует два способа запросить информацию о наличии мест:

  • Получение информации о наличии мест для определенного рейса
  • Получение информации о наличии мест для всех рейсов по заданному направлению

Получение информации о наличии мест для определенного рейса

Для получения информации о наличии мест для определенного рейса необходимо указать в запросе:

  • /OTA_AirAvailRQ/OptionalQualifiers/FlightQualifiers/VendorPrefs/Airline/@Code — код маркетингового перевозчика
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/@DepartureDateTime — дата отправления
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/@FlightNumber — номер рейса
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/OriginLocation/@LocationCode — код пункта отправления
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/DestinationLocation/@LocationCode — код пункта назначения

Пример запроса
Копировать 
<OTA_AirAvailRQ ReturnHostCommand="true" Version="2.4.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <OptionalQualifiers>
    <FlightQualifiers>
      <VendorPrefs>
        <Airline Code="SU"/>
      </VendorPrefs>
    </FlightQualifiers>
  </OptionalQualifiers>
  <OriginDestinationInformation>
    <FlightSegment DepartureDateTime="08-31" FlightNumber="2578">
      <DestinationLocation LocationCode="LON"/>
      <OriginLocation LocationCode="MOW"/>
    </FlightSegment>
  </OriginDestinationInformation>
</OTA_AirAvailRQ>

Получение информации о наличии мест для всех рейсов по заданному направлению

Для получения информации о наличии мест для всех рейсов по заданному направлению:

  • /OTA_AirAvailRQ/OptionalQualifiers/FlightQualifiers/VendorPrefs/Airline/@Code — код маркетингового перевозчика
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/@DepartureDateTime — дата отправления
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/OriginLocation/@LocationCode — код пункта отправления
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/DestinationLocation/@LocationCode — код пункта назначения

Дополнительно в запросе можно указать:

  • /OTA_AirAvailRQ/OptionalQualifiers/FlightQualifiers/@DirectOnly — значение true позволит искать только прямые рейсы
  • /OTA_AirAvailRQ/OptionalQualifiers/FlightQualifiers/@OnlineOnly — значение true позволит искать только рейсы с одним маркетинговым перевозчиком (исключить интерлайн перевозки)
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/ConnectionLocations/CarrierSpecificDisplay/LegTwo/VendorPrefs/@ExcludeCodeshares — значение true позволит исключить кодшейринговые рейсы
  • /OTA_AirAvailRQ/OptionalQualifiers/TimeQualifiers/DepartureTime/@WindowAfter и @WindowBefore — показать рейсы с вылетом в указанный временной промежуток
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/ConnectionLocations/ConnectionLocation/@LocationCode — код пункта пересадки
  • /OTA_AirAvailRQ/OriginDestinationInformation/FlightSegment/ConnectionLocations/ConnectionLocation/ConnectTimeOverrides/@Min и @Max — указать минимальное и максимальное время стыковки

Пример запроса
Копировать 
<OTA_AirAvailRQ ReturnHostCommand="true" Version="2.4.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10">
  <OptionalQualifiers>
    <FlightQualifiers DirectOnly="true">
      <VendorPrefs>
        <Airline Code="SU"/>
      </VendorPrefs>
    </FlightQualifiers>
  </OptionalQualifiers>
  <OriginDestinationInformation>
    <FlightSegment DepartureDateTime="08-31T00:00">
      <DestinationLocation LocationCode="LED"/>
      <OriginLocation LocationCode="MOW"/>
    </FlightSegment>
  </OriginDestinationInformation>
</OTA_AirAvailRQ>
