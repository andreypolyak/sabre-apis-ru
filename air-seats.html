Чтение бронирования (TravelItineraryReadRQ)

Сервис TravelItineraryReadRQ позволяет открыть бронирование в текущей сессии и прочесть его содержимое.

В качестве параметра UniqueID/@ID в запросе указывается код бронирования (PNR Record Locator).

В результате чтения бронирования должен быть составлен список сегментов, для которых необходимо получить карту мест.

Пример запроса

<TravelItineraryReadRQ Version="3.8.0" xmlns="http://services.sabre.com/res/tir/v3_8">
  <MessagingDetails>
    <SubjectAreas>
      <SubjectArea>FULL</SubjectArea>
    </SubjectAreas>
  </MessagingDetails>
  <UniqueID ID="SKJIYN"/>
</TravelItineraryReadRQ>

Получение карты мест (EnhancedSeatMapRQ)

Для получения карты мест используется сервис EnhancedSeatMapRQ.

В запросе необходимо указать:

  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/RequestType — тип запроса, всегда Payload
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/Flight/@origin и /@destination — код аэропортов вылета и прилета
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/Flight/DepartureDate — дата вылета
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/Flight/Marketing — номер рейса
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/Flight/Marketing/@carrier — код маркетингового перевозчика
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/CabinDefinition/RBD — класс бронирования
  • /EnhancedSeatMapRQ/SeatMapQueryEnhanced/POS/PCC — iPCC

Бронирование места в салоне (AirSeatLLSRQ)

Для бронирования места в салоне используется сервис AirSeatLLSRQ.

В запросе необходимо указать:

  • /AirSeatRQ/Seats/Seat/@NameNumber — номер пассажира
  • /AirSeatRQ/Seats/Seat/@Number — номер места
  • /AirSeatRQ/Seats/Seat/@SegmentNumber — номер сегмента

Обратите внимание на то, что бронирование места в салоне возможно только для одного человека на одном рейсе. Для бронирования мест в салоне для нескольких пассажиров и (или) на нескольких сегментах, необходимо отправить на бронирование места для каждой пары пассажир — рейс.

Пример запроса

<AirSeatRQ ReturnHostCommand="true" Version="2.0.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <Seats>
    <Seat NameNumber="1.1" Number="25A" SegmentNumber="1"/>
  </Seats>
</AirSeatRQ>

<AirSeatRS Version="2.0.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

Сохранение бронирования (EndTransactionLLSRQ)

Для сохранения изменений в бронировании необходимо отправить запрос к сервису EndTransactionLLSRQ. В результате выполнения запроса все изменения в бронировании, выполненные в текущей сессии, будут сохранены, а бронирование закрыто.

В запросе нужно указать код ReceivedFrom: он будет отображаться в истории бронирования.

Пример запроса

<EndTransactionRQ ReturnHostCommand="true" Version="2.0.6" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <EndTransaction Ind="true"/>
  <Source ReceivedFrom="SWS"/>
</EndTransactionRQ>

<EndTransactionRS Version="2.0.6" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

Игнорирование бронирования (IgnoreTransactionLLSRQ)

В некоторых случаях при попытке сохранения бронирования может возникнуть ошибка. Чаще всего они возникают из-за того, что в момент между открытием и сохранением бронирования оно было изменено, например, если перевозчик отправил сообщение с правилами выкупа билета (тайм-лимитом). В этом случае внесенные изменения не будут сохранены и все запросы необходимо повторить, предварительно проигнорировав текущее бронирование отправкой запроса к сервису IgnoreTransactionLLSRQ.

Пример запроса

<IgnoreTransactionRQ xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" ReturnHostCommand="true" Version="2.0.0"/>

Пример ответа

<IgnoreTransactionRS Version="2.0.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:stl="http://services.sabre.com/STL/v01" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <stl:ApplicationResults status="Complete">
    <stl:Success timeStamp="2016-07-31T14:28:13-05:00">
      <stl:SystemSpecificResults>
        <stl:HostCommand LNIATA="0F0162">I</stl:HostCommand>
      </stl:SystemSpecificResults>
    </stl:Success>
  </stl:ApplicationResults>
</IgnoreTransactionRS>

Оформление EMD для платных мест

Для платных мест обязательно требуется оформление EMD. Обратите внимание на то, что EMD должны быть оформлены в бронировании после билетов.

Процесс оформления EMD подробно описан в разделе Оформление билетов и EMD.

Удаление забронированного места (AirSeatCancelLLSRQ)

Для удаления забронированного места используется сервис AirSeatCancelLLSRQ.

В запросе необходимо указать:

  • для удаления всех забронированных мест:
    • /AirSeatCancelRQ/Seats/Seat/@All — значение true
  • для удаления всех забронированных мест на рейсах определенного перевозчика:
    • /AirSeatCancelRQ/Seats/Seat/VendorPrefs/Airline/@Code — код перевозчика
  • для удаления всех забронированных мест на определенном сегменте:
    • /AirSeatCancelRQ/Seats/Seat/@SegmentNumber — номер сегмента в бронировании
  • для удаления определенного забронированного места на определенном сегменте:
    • /AirSeatCancelRQ/Seats/Seat/@SegmentNumber — номер сегмента в бронировании
    • /AirSeatCancelRQ/Seats/Seat/@SeatNumber — номер места
  • для удаления определенного забронированного места для определенного пассажира:
    • /AirSeatCancelRQ/Seats/Seat/@SegmentNumber — номер сегмента в бронировании
    • /AirSeatCancelRQ/Seats/Seat/@NameNumber — номер пассажира в бронировании

Пример запроса

<AirSeatCancelRQ ReturnHostCommand="true" Version="2.1.0" xmlns="http://webservices.sabre.com/sabreXML/2011/10" xmlns:xs="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <Seats>
    <Seat All="true"/>
  </Seats>
</AirSeatCancelRQ>
